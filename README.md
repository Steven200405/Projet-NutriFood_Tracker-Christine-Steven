# NutriFood_Tracker

NutriFood_Tracker est une application web qui vous aide à mieux comprendre vos habitudes alimentaires, à consulter des produits, et à obtenir des recommandations d’alternatives plus saines grâce au Nutri-Score.

API utilisée : Open Food Facts (https://fr.openfoodfacts.org)

---

## Fonctionnalités

- **Authentification**
  - Création de compte
  - Connexion / Déconnexion
- **Profil utilisateur**
  - Créer / lire / modifier un profil
- **Questionnaire habitudes alimentaires**
  - Évalue vos habitudes alimentaires
  - Calcule un **Nutri-Score moyen** (A, B, C, D, E)
- **Recherche produit**
  - Recherche via une barre de recherche (produits Open Food Facts)
  - Affichage des informations clés (ex. Nutri-Score, allergies, etc.)
- **Historique**
  - Visualiser l’historique des questionnaires complétés
- **Recommandations**
  - Conseils d’alternatives **avec un meilleur Nutri-Score** quand c’est possible
